Task Manager App

A simple Task Manager web application built with Node.js, Express, MongoDB, and EJS.
This app allows users to create, view, edit, and delete tasks easily through a simple interface.

Features

Add new tasks with a title, description, and status.

View a list of all tasks.

Edit existing tasks.

Delete tasks.

Status badges to indicate task progress: Pending, In Progress, Completed.

Clean and simple user interface using EJS and Bootstrap.

Tech Stack

Backend: Node.js, Express.js

Database: MongoDB (with Mongoose ODM)

Frontend: EJS (Embedded JavaScript templates), Bootstrap

Development Tools: Nodemon for auto-reloading during development
